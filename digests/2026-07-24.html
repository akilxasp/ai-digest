FROM DEMO TO DAILY DRIVER
The through-line today is agents leaving the sandbox: Claude's voice mode now reschedules meetings, OpenAI's Presence puts agents in front of enterprise data with an escalation path to a human, and Gemini 3.6 Flash is tuned for cheaper multi-step tool use rather than raw benchmark scores. Underneath it all, the Moonshot accusation is a reminder of how much the underlying weights are now worth fighting over.

Daily Tip
If you're wiring an agent to real tools this week, don't grade it with a single LLM judge — a community eval built this week found judge agreement as low as 22% on identical transcripts.
